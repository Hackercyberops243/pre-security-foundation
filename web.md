# How the Web Works

Web communication happens between a client (browser) and a server.

## Key Process

1. DNS resolves a domain name into an IP address  
2. The browser sends an HTTP/HTTPS request  
3. The server processes it  
4. The response is sent back to the browser  

## Important Concepts

- HTTP methods (GET, POST)  
- Headers and responses  
- Frontend vs backend  

## Why It Matters in Cybersecurity

Most modern attacks target web applications, so understanding web communication is essential.
