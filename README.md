# pre-security-foundation
Foundation of cybersecurity

This repository summarizes the key concepts I learned after completing the Pre Security path and SECO certification on TryHackMe.

The goal of this project is to document my understanding of the core technologies behind cybersecurity before moving into more advanced topics.

## Topics Covered

- Networking fundamentals  
- How the web works (DNS, HTTP)  
- Computer and operating systems basics  
- Security principles (CIA triad, cryptography)  
- Offensive vs defensive mindset  

## Why This Project

Completing labs is useful, but I wanted to turn my learning into something structured and shareable.

This repository reflects my journey toward becoming a cybersecurity professional with both offensive and defensive skills (Purple Team).
