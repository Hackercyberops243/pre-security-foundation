# Security Fundamentals

Cybersecurity protects systems, networks, and data.

## CIA Triad

- Confidentiality → protect data  
- Integrity → prevent alteration  
- Availability → ensure access  

## Cryptography

Used to secure data through:
- Encryption  
- Hashing  

## Offensive vs Defensive Security

- Offensive → finding vulnerabilities  
- Defensive → protecting and monitoring systems  

## Mindset

Understanding both attacker and defender perspectives is key to becoming a well-rounded cybersecurity professional.
