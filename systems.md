# Computer and Operating Systems Basics

Computers use hardware and software to process data.

## Key Components

- CPU → processes instructions  
- RAM → temporary memory  
- Storage → permanent data  

## Operating Systems

- Windows and Linux manage system resources  
- CLI (command line) allows advanced control  

## Additional Concepts

- Virtualization → multiple systems on one machine  
- Cloud → remote infrastructure  

## Why It Matters in Cybersecurity

Attackers often exploit system misconfigurations. Understanding systems is essential for both attack and defense.
